# Video Call App with WebRTC and Supabase

A browser-based peer-to-peer video calling application built with React, WebRTC, and Supabase Realtime.

## Live Demo

https://your-app.vercel.app

## Features

- Create and join private rooms
- Real-time video and audio calling
- Responsive UI
- Supabase-powered signaling

## Tech Stack

- Vite-React
- WebRTC
- Supabase
- Tailwind CSS

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
